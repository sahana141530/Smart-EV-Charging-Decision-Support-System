# ⚡ Smart EV Charging, Cost Analysis and Decision Support System

## 📌 Project Overview

This project is a web-based application designed to help Electric Vehicle (EV) users make intelligent decisions related to charging, travel planning, cost optimization, and environmental impact.
It integrates multiple modules such as cost calculation, range estimation, environmental analysis, smart decision-making, vehicle comparison, and map-based charging station guidance into one unified system.

---

## 🚀 Key Features

### ⚡ Charging Cost Calculator
- Calculates energy required and charging cost  
- Suggests optimized charging strategies  

### 🚗 Range Estimation
- Predicts whether a trip is feasible based on battery level  
- Provides travel safety insights  

### 🌱 Environmental Impact Analysis
- Calculates CO₂ reduction and cost savings  
- Shows environmental benefits of EV usage  

### 🧠 Smart Decision System
- Provides intelligent recommendations based on:
  - Battery level  
  - Distance  
  - Charging needs  

### ⚖ EV vs Petrol/Diesel Comparison
- Compares:
  - Cost
  - Monthly & yearly savings
  - CO₂ emissions  
- Helps users choose the most economical option  

### 📍 Location-Based Charging Station Finder
- Detects user location  
- Shows nearby charging stations  
- Displays route to selected station  

### 🎯 Smart Trip Advisor (Standout Feature)
- Combines:
  - Battery level  
  - Distance  
  - Charging availability  
- Gives complete travel recommendation  

### 📊 Interactive Dashboard
- Charts and visualization using Chart.js  
- Real-time updates  

### 📜 History Tracking
- Stores past calculations  
- Helps analyze usage patterns  

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Visualization: Chart.js  
- Mapping: Leaflet.js (OpenStreetMap)  
- Database: In-memory / optional
